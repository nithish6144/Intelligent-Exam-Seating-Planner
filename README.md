# Intelligent Exam Seating Planner

A robust, automated seating arrangement system for educational institutions.

## Features
-   **Dynamic Room Management**: Configure custom room sizes and layouts.
-   **Intelligent Allocation**: Prevents adjacent students from having the same subject.
-   **Exam Wizard**: Guided step-by-step process to conduct exams.
-   **PDF Reporting**: Generates professional seating charts for each room.

## Installation
1.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
2.  Run the application:
    ```bash
    python app.py
    ```
3.  Open `http://localhost:5000`
